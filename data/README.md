# Sample Dictionaries

This folder contains example data files you can import from Options → Offline Readings.

- `en_US.txt` — English IPA dictionary (TSV: `word\t/ipa/`, may contain comma-separated variants)
- `jmdict-eng-3.6.1.json` — JMdict mini dump (top-level `{ words: [...] }`)

Note: these files are not bundled into the extension package; they are for convenience when testing.
